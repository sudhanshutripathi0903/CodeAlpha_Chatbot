# Basic Chatbot using Python

This is a simple rule-based chatbot created using Python. 
The chatbot responds to basic user inputs like greetings, name queries and exit commands.

## Features
- Interactive text-based chat
- Predefined responses
- Randomized replies
- Exit command support

## Technologies Used
- Python

## How to Run
1. Install Python
2. Save the file as chatbot.py
3. Run command:
   python chatbot.py

## Sample Inputs
hi  
hello  
how are you  
name  
bye  

## Output
Chatbot gives appropriate responses according to user input.
